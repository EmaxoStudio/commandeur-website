## DESIGN SYSTEM

### STYLE
- dark premium
- ruhig, seriös, hochwertig
- kein Marketing-Stil

### FARBEN
- navy / schwarz
- weißer Text
- gedämpftes blau für CTA

### TYPOGRAFIE
- klare Sans-Serif
- starke Hierarchie
- große Headlines

### LAYOUT
- große Abstände
- klare Sections
- mobile first
